Quiz Whiz Game

Its time to play quiz.Let's play quiz game!!

  1. Click the start button to start the quiz game
  2. There are 10 random questions with four option
  3. You need to select one option
  4. Each correct answer gives you a point
  5. When you click any option button it will pop with alert saying whether its correct or incorrect answer
  6. End of the game your score will be displayed and you can Try Again
